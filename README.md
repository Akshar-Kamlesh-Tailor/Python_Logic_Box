# 🚀 Pattern Generator & Number Analyzer

A beginner-friendly **Python console application** that demonstrates control flow, loops, pattern generation, conditional logic, and menu-driven programming. It is a clean mini-project suitable for showcasing Python fundamentals.

## ✨ Features
- ⭐ Menu-driven interface
- 🔺 Generate a right-angle star pattern
- 🔢 Analyze a range of numbers
  - Detect Even/Odd
  - Calculate total sum
- 🚪 Graceful exit option

## 🛠️ Technologies
- Python 3

## 📊 Flowchart

```text
Start
  |
  v
Display Menu
  |
  +--> 1 -> Input rows -> Print Pattern -> Menu
  |
  +--> 2 -> Input start/end -> Check Even/Odd -> Sum Numbers -> Menu
  |
  +--> 3 -> Exit
```
